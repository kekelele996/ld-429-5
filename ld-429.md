## 项目主要需求

### 项目概述

【Web前端】画廊虚拟导览平台。纯前端 3D 虚拟画廊体验，用户可在浏览器中漫步虚拟展厅、查看作品大图、切换展览、收听语音导览（模拟），支持多人在线状态显示（模拟）。数据存储于 localStorage / IndexedDB。**本项目用于验证模型跨文件协同改动能力，禁止将多个职责合并进单一文件。**

### 核心实体（≥ 3 个数据模型）

1. **展厅（GalleryRoom）** — 字段：展厅ID、展厅名称、展厅尺寸（宽×高×深）、墙壁颜色、地板材质、灯光配置（亮度、色温、聚光灯位置列表）、作品挂载点数组（位置+角度+尺寸）、展厅类型（Main/Side/Virtual/Outdoor）
   - 定义位置：`types/room.ts`

2. **作品（Artwork）** — 字段：作品ID、标题、艺术家名、创作年份、媒介、图片URL、缩略图URL、描述、展厅ID、挂载位置坐标、画框样式（None/Simple/Classic/Modern/Ornate）、标签列表
   - 定义位置：`types/artwork.ts`

3. **展览（Exhibition）** — 字段：展览ID、展览标题、策展人、展览描述、封面图、展厅ID列表（参观顺序）、参观路线路径点数组、音频导览URL列表（可选）、状态（Active/Upcoming/Past）
   - 定义位置：`types/exhibition.ts`

4. **导览标注（GuideAnnotation）** — 字段：标注ID、关联作品ID、标注位置（3D坐标）、标题、描述、音频URL（可选）、排序
   - 定义位置：`types/guide.ts`

5. **访客记录（VisitorLog）** — 字段：访客ID（匿名）、进入时间、停留时长、浏览作品ID列表、当前展厅ID、在线状态（InGallery/Left）
   - 定义位置：`types/visitor.ts`

### 核心页面（≥ 4 个）

1. **3D 画廊漫游** `/gallery` — 消费：GalleryRoom + Artwork + GuideAnnotation
   - Three.js 第一人称视角在虚拟画廊中行走，靠近作品时弹出信息卡片和导览标注，支持WASD/方向键移动、鼠标旋转视角
   - 使用 `<ArtworkInfoCard>`, `<GuideTooltip>`, `<MiniMap>` 共享组件

2. **展览列表** `/exhibitions` — 消费：Exhibition + GalleryRoom
   - 展览卡片网格，点击进入画廊漫游并自动导航到该展览的展厅
   - 使用 `<ExhibitionCard>`, `<StatusBadge>`, `<EmptyState>`

3. **作品详情** `/artwork/:id` — 消费：Artwork + GuideAnnotation
   - 作品大图查看（支持缩放和拖拽）、作品信息面板、导览标注列表
   - 使用 `<ArtworkInfoCard>`, `<GuideTooltip>`

4. **展厅编辑器** `/editor` — 消费：GalleryRoom + Artwork
   - 可视化编辑展厅布局：调整墙壁颜色、灯光参数、拖拽放置作品到挂载点、预览3D效果
   - 使用 `<PropertyPanel>`, `<DraggableItem>`

5. **参观统计** `/analytics` — 消费：VisitorLog + Artwork
   - 参观热力图（作品停留时间）、最受欢迎作品排行、参观路线回放
   - 使用 `<StatCard>`, `<HeatmapChart>`

### 横切关注点

1. **全局异常处理**
   - 触达文件层：Three.js渲染异常 + 图片加载失败 + IndexedDB异常 + ErrorBoundary + 错误类型

2. **主题切换（深色/浅色）**
   - 触达文件层：CSS变量 + themeStore + Three.js场景光照切换 + 主题类型定义

### 共享枚举/常量（≥ 2 组）

1. **展厅类型 RoomType**：Main / Side / Virtual / Outdoor
   - 前端：`types/enums.ts`

2. **展览状态 ExhibitionStatus**：Active / Upcoming / Past

3. **画框样式 FrameStyle**：None / Simple / Classic / Modern / Ornate

4. **默认灯光配置 DEFAULT_LIGHTING** — 各展厅类型的默认灯光参数
   - 前端：`constants/lighting.ts`

### 共享前端组件清单（≥ 3 个）

1. **`<ArtworkInfoCard>`** — 3D画廊漫游、作品详情
2. **`<GuideTooltip>`** — 3D画廊漫游（导览标注气泡）、作品详情
3. **`<ExhibitionCard>`** — 展览列表
4. **`<MiniMap>`** — 3D画廊漫游（俯视小地图）
5. **`<PropertyPanel>`** — 展厅编辑器（属性调节面板）

### 共享 hooks/utils 清单（≥ 2 个）

1. **`useFirstPersonController`** — 封装第一人称视角控制器（WASD移动、鼠标旋转、碰撞检测）
2. **`useGalleryScene`** — 封装展厅场景初始化（灯光、墙壁、地板、作品挂载）
3. **`threeUtils/`** — 包含材质工厂、纹理加载器、光照配置器

### 技术要求

- **前端端口**：18809
- **前端**：React 18 + TypeScript + Vite
- **3D引擎**：Three.js + @react-three/fiber + @react-three/drei + @react-three/rapier（碰撞检测）
- **UI 库**：Tailwind CSS
- **状态管理**：Zustand
- **本地存储**：IndexedDB（Dexie.js）
- **路由**：React Router v6
- **构建工具**：Vite
